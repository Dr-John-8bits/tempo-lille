# Code de conduite

## Notre engagement
Nous voulons un espace de collaboration respectueux, inclusif et orienté solution.

## Comportements attendus
- Être courtois et constructif dans les échanges.
- Donner des retours techniques factuels, sans attaque personnelle.
- Accepter les points de vue différents et chercher le compromis.

## Comportements inacceptables
- Harcèlement, intimidation, insultes, discrimination.
- Propos haineux, sexistes, racistes, homophobes ou dégradants.
- Publication d'informations privées sans consentement.

## Signalement
En cas de problème:
- Ouvre une issue si le sujet peut être traité publiquement.
- Sinon, contacte directement les mainteneurs pour un traitement discret.

Les mainteneurs se réservent le droit de modérer, éditer ou supprimer tout contenu contraire à ce code.
