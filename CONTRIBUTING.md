# Contribuer à Tempo Lille

## Pré-requis
- HTML/Bootstrap/JS simple. Aucune build step.
- Respecter le mobile-first et le dark mode auto.

## Lancer en local
- VS Code + Live Server, ou `python3 -m http.server`.

## Style de code
- HTML sémantique, ARIA quand pertinent.
- JS vanilla (pas de dépendances lourdes).
- CSS minimal dans la page (ou futur `assets/` commun si besoin).

## Commits
- **Conventional Commits** : `feat:`, `fix:`, `docs:`, `refactor:`, `chore:`.
- Une PR = une intention claire, description concise, capture si visuel.

## Tests manuels
- Vérifier sur mobile (petites largeurs).
- Tester sans géoloc / permissions refusées.
- Vérifier accessibilité de base (contrastes, focus, aria-labels).

Merci !
